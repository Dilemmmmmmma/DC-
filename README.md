# DC聊天机器人多号版
一个Discord 机器人，可以在多个频道根据设定时间发送消息，还支持多个频道。

## 运行截图

![Proof](https://i.ibb.co/vQkwwmb/We-Chat-20241212200238.jpg)
## 有什么用

- 帮你保频道活跃度
- 根据你设置的时间段在某频道发送消息
- 根据你设定频道的最近100条信息随机选择一条发送
- 支持多个token
- 支持多个频道
- 95%的代码来自Chatgpt和Claude

## 需要工具

- 手
- Python3.11
